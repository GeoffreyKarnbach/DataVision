A Simple, Powerful Workflow

From raw data to actionable insights in four logical steps.

1. Register Things

Start by registering your raw data sources. Explicitly define each "Thing" and its MQTT topic to build a master list of all available data streams.

2. Model Environment

Visually construct a hierarchical model of your physical space. Create buildings, floors, and rooms, then assign Things to give them rich contextual meaning.

3. Deploy Pipeline

With one click, compile your model into an executable data pipeline. This automatically generates Node-RED flows and deploys the Virtual Data Fabric runtime.

4. Visualize & Interact

Explore your live system through an interactive knowledge graph. Click any node to see real-time plots of the corresponding sensor data instantly.

A Foundation for Intelligence

Unified Namespace

The core principle. Data is transformed from cryptic topics into a clean, human-readable hierarchy—a single source of truth for all services.

Living Knowledge Graph

Your system's structure isn't just a diagram; it's a queryable Neo4j graph database, providing the backbone for powerful relational queries and a rich UI.

Extensible Service Ecosystem

A modular microservices approach. Specialized backends for plotting and graph queries decouple the UI from data sources, making the system easy to extend.
